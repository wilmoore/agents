# LinkedIn Inbound Engine ChatGPT Project

## Summary

Added a new ChatGPT project for LinkedIn demand capture and conversion.

## Changes

- New ChatGPT project: `chatgpt/projects/g-p-698c66f0c2608191ac76fe4366c82666/instruction.md`
- Project purpose: Capture, qualify, and convert inbound demand from LinkedIn
- Features routing system with numeric options for lead qualification
- Covers two entry vectors: Profile Pull (connection requests) and Thread Injection (comments)
